If You Really Need a Single File

If you absolutely need a single HTML file, you have two options:

Option 1: Convert to Direct HTML Injection (Recommended for bundling)

This requires modifying your main.html to load pages via fetch/insertHTML instead of iframes. I can help you create this version.

Option 2: Use the Bundler Script

The bundle_presentation.py script will attempt to bundle everything, but:

  • Media files (images/videos) will still be external references
  • To embed media, use --embed-media flag (creates 50-100+ MB files)
  • Complex iframe-based structure may not work perfectly

Quick Start - Recommended Approach

For GitHub Pages (5 minutes):

git init
git add .
git commit -m "Initial presentation"
# Create repo on GitHub, then:
git remote add origin https://github.com/YOUR_USERNAME/repo-name.git
git push -u origin main
# Enable GitHub Pages in repo settings
# Your presentation: https://YOUR_USERNAME.github.io/repo-name/main.html

For Netlify Drop (2 minutes):

  1. Go to app.netlify.com/drop
  2. Drag your entire presentation folder
  3. Get instant URL!

For Local Testing:

# Python
python -m http.server 8000

# Node.js
npx http-server -p 8000

# Then visit: http://localhost:8000/main.html
